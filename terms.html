Revisions

All packages include up to 3 rounds of revisions following the initial preview. A "round" means one set of consolidated feedback responded to with one updated version of the site.

What counts as a revision

  • Changes to text, colours, fonts, or layout
  • Adding or removing content elements
  • Adjustments to images or image placement
  • Changes to page structure or navigation

What does not count as a revision

  • Fixing genuine errors or bugs caused by us
  • Changes required due to our misunderstanding of the agreed brief
  • Minor text corrections (spelling, factual errors)

We strongly recommend consolidating all feedback into a single response per round. Piecemeal or drip-fed feedback may use up revision rounds more quickly. Each time you send a set of changes and we implement them, that constitutes one revision round.

Additional revisions

If you require changes beyond the 3 included rounds, additional revisions are charged at £50 per hour, billed in 30-minute increments. We will always confirm the estimated time and cost before proceeding.

Scope changes

If you request changes that significantly alter the scope of the project from the original brief — such as adding new pages, new features, or major structural redesign — this may constitute a new quotation rather than a revision. We will advise you clearly if this is the case.
